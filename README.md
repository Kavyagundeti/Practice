## 🚀 Featured Project

### 📦 CI/CD Pipeline for Online Book Store

> Automated build & deployment pipeline using **Jenkins + Maven + GitHub + Apache Tomcat + AWS EC2**

| Tool | Purpose |
|------|---------|
| Jenkins | CI/CD orchestration & pipeline automation |
| Maven | Build tool — generates WAR artifacts |
| GitHub | Version control & webhook triggers |
| Apache Tomcat | Web application server |
| AWS EC2 | Cloud hosting on Linux instance |

- ✅ Automated build triggered on every GitHub push
- ✅ WAR file generated via Maven and deployed to Tomcat
- ✅ Reduced manual deployment effort to zero
